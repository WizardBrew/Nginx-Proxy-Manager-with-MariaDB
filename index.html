🚀 NGINX Setup Guide (Ubuntu)

Install Service Server Block Reverse Proxy SSL

Step 1: Install NGINX

Update packages and install NGINX:

sudo apt update
sudo apt install nginx -y

Step 2: Manage Service

Command Description Example
systemctl start nginx Start NGINX service sudo systemctl start nginx
systemctl enable nginx Enable on boot sudo systemctl enable nginx
nginx -t Test configuration sudo nginx -t

Step 3: Create Server Block

Create a file in /etc/nginx/sites-available/example:

server {
    listen 80;
    server_name example.com www.example.com;
    root /var/www/example;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}

Enable it:

sudo ln -s /etc/nginx/sites-available/example /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

Step 4: Reverse Proxy

Forward traffic to a backend app (Node.js on port 3000):

server {
    listen 80;
    server_name app.example.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

Step 5: Secure with SSL

Install Certbot and generate certificates:

sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d example.com -d www.example.com
sudo certbot renew --dry-run
